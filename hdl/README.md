Instructions:

| code  | name
| ----: | :--------------
| `000` | no-op
| `001` | flip bit
| `010` | move tape left
| `011` | move tape right
| `100` | left bracket
| `101` | right bracket
| `110` | print
| `111` | input

CPU output codes:

| code | name
| ---: | :-----------
| `00` | default/null
| `01` | used input
| `10` | print 0
| `11` | print 1
